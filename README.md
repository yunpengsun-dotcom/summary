# summary
We assembled a high‑quality draft genome of Enterococcus faecalis (N50 = 2.76 Mb, 9 contigs), annotated 3,118 protein‑coding genes, and confirmed strong homology to a reference strain (91.6% identity). RNA‑seq data were processed with Trimmomatic and FastQC, then aligned to the assembly (~87% assignment for BH, ~63% for serum). DESeq2 identified 1,336 significantly differentially expressed genes (|log2FC|>1, padj<0.05) between serum and rich medium (667 up, 670 down). This study provides a solid genomic resource and reveals extensive transcriptional reprogramming of E. faecalis under serum‑like conditions
